1) Созлаем папку пректа, потом проект django-admin startproject locallibrary .    ## locallibrary - имя проекта

2. Создаем приложение py -3 manage.py startapp blog			## blog - имя приложения 
(команду необходимо выполнять из папки, в которой находится manage.py )

3) прописываем приложение в разделе INSTALLED_APPS файла settings.py
Добавляем строку catalog.apps.CatalogConfig      ## Catalog - имя придлжения

4) Добавляем код в urls.py в корне проекта

# Используйте include() чтобы добавлять URL из каталога приложения
from django.urls import include
from django.urls import path
urlpatterns += [
     path('blog/', include('blog.urls')),
]

6) Вводим команды из корня проекта
py -3 manage.py makemigrations
py -3 manage.py migrate
py -3 manage.py runserver 	


7) Создаем суперпользователя 
py -3 manage.py createsuperuser
